# Онлайн журнал сотрудников

Индивидуальный проект, где пользователи могут создавать, редактировать и удалять своих сотрудников. Веб-приложение поддерживает многопользовательскую систему с авторизацией.

## Технологии:

- React
- Redux toolkit
- JavaScript
- Node js
- express
- SQLite
- Prisma |
- tailwind CSS
- TypeScript
